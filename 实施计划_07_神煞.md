# 神煞模块(本命神煞)Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 给 bazi-core 加一个确定性的「本命神煞」结构事实层——从四柱(年月日时干支)按单一流派起例查表,算出命中哪些神煞、落在哪柱,作为**纯描述性结构料**进 `ChartAnalysis.神煞` + cite 白名单,**不碰 favorSign**(L3/L4 单一吉凶真相源不动),吉凶/含义解读全交 LLM。

**Architecture:** 照现有 facts 模块范式(`roots.ts`/`combos.ts`):新建 `src/facts/shensha.ts` + 起例常量 `src/constants/shensha-tables.ts`;`detectShensha(four, dm, opts?)` 按起例基准分四组纯函数(byStem / byYearBranch / byMonthBranch / byPillar)+ 少量 special(元辰/天罗地网/学堂词馆),组装成 `ShenshaResult`;插进 `analyzeChart` 的 detectRelations 之后;cite 走 `extractCitableStructures`。全部确定性查表、无 RNG。

**Tech Stack:** TypeScript (ESM) · vitest · zod(本模块不新增 schema)· 复用 `src/constants/gan-zhi.ts`(干支/五行/阴阳)与 `src/facts/relations.ts`(`DIZHI_SANHE`/`DIZHI_SANHUI`)。

**流派拍板(已与作者确认·锁死单流派):**
- 三合局类(桃花/驿马/华盖/将星/劫煞/灾煞/亡神)+ 孤辰寡宿 = **年支起**(忠《三命通会》古法)。
- 十二长生 **火土同宫**(戊≡丙、己≡丁)——但本计划用**直接查表**(研究已给每神煞确切目标),不实现长生推导。
- flag 默认:**羊刃只五阳干有刃**(阴干无)、**魁罡=庚辰/壬辰/戊戌/庚戌**(非壬戌)、**孤辰寡宿基准=年支**。
- 定位:神煞=结构事实(干支→present),**绝无「神煞→性格词」表**;不进 favorSign。

```typescript
  const out: PillarSlot[] = [];
```

```typescript
// 顶部 import 加: import { GAN_YINYANG } from '../constants/gan-zhi';
export interface DetectShenshaOpts {
  gender?: '乾' | '坤';   // 元辰需要;不传则元辰跳过 + caveat
}

function specialYuanchen(four: FourPillars, opts: DetectShenshaOpts, caveats: string[]): ShenshaHit[] {
  const yz = four.年?.zhi;
  const yg = four.年?.gan;
  if (!yz || !yg) return [];
  if (!opts.gender) { caveats.push('元辰:缺性别,未计算'); return []; }
  const yearYang = GAN_YINYANG[yg] === '阳';
  // 阳男 或 阴女 → 用"阳男阴女"表;阴男 或 阳女 → "阴男阳女"表
  const isMale = opts.gender === '乾';
  const useYangNan = (isMale && yearYang) || (!isMale && !yearYang);
  const target = useYangNan ? T.元辰_阳男阴女[yz] : T.元辰_阴男阳女[yz];
  const pos = branchesAt(four, [target]);
  return pos.length ? [mk('元辰', '年支性别', pos)] : [];
}

export function detectShensha(four: FourPillars, dm: DayMaster, opts: DetectShenshaOpts = {}): ShenshaResult {
  const caveats: string[] = [];
  if (!four.时) caveats.push('缺时柱:时柱相关神煞可能漏判');
  const hits = [
    ...byStem(four, dm),
    ...byYearBranch(four),
    ...byMonthBranch(four),
    ...byPillar(four),
    ...specialYuanchen(four, opts, caveats),
  ];
  return { hits, caveats };
}
```
