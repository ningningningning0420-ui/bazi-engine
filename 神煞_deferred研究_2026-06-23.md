## 5. 天罗地网 ✅ 直接锁(纳音条件版=《三命通会》正法)

三层:
1. **门槛(年柱纳音)**:火→天罗候选;水/土→地网候选;金/木→无煞不扫。
2. **触发(四柱地支)**:火命见 戌/亥 → 天罗;水土命见 辰/巳 → 地网。两煞地支互斥。
3. **男女(命中后主忌标记,非命中条件)**:男忌天罗、女忌地网。已写进 META gist,引擎级**无需性别即可判命中**。

```
火 → {煞:'天罗', 触发支:['戌','亥']}
水 → {煞:'地网', 触发支:['辰','巳']}
土 → {煞:'地网', 触发支:['辰','巳']}
金/木 → 无
```

- basis = '纳音'。**边界澄清**:本煞按纳音五行分类,与火土同宫(长生学说)**正交**——火命(炉中火)≠土命(城头土),不因火土同宫合并。**故天罗地网不受 §6 拍板影响。**
- 排除:无条件今法(凑齐辰巳戌亥)/ 辰戌简化版。置信 high,**不需拍板**。

## 6. ★ 唯一拍板项:学堂/词馆 的「土」同宫

金/木/水/火 两套体系完全一致,只有**土**分歧:

| | A:水土同宫(忠于典籍) | B:火土同宫(服从模块全局) |
|---|---|---|
| 学堂(土) | 长生 **申** | 长生 **寅** |
| 词馆(土) | 临官 **亥** | 临官 **巳** |
| 依据 | 《三命通会·论学堂词馆》本章原文「水土长生申,临官亥」 | 模块全局十二长生已锁火土同宫(戊≡丙/己≡丁) |
| 代价 | 学堂/词馆单存一张纳音长生/临官轮(与排盘火土同宫数组分开) | 偏离本章典籍原意,属「自创对齐」需注释 |

两校验相反推荐(学堂校验→A / 词馆校验→B);综合官倾向 **A + 例外标注**(理由:学堂词馆是本章专有禄命法,典籍明用水土同宫;模块已有 `ShenshaBasis='纳音'`,技术上单存一表可行不破坏其他神煞;红线=命理正确+宁缺勿错)。**只影响土命盘,金木水火命两版一致。**
